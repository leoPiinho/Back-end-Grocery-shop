MyGrocery API

Esta é uma API REST desenvolvida com Spring Boot para a gestão de itens de um mercado. O projeto permite realizar operações completas de CRUD (Create, Read, Update, Delete) em produtos, com persistência de dados em um banco de dados MySQL.

---------------------------------------------------------------------------------------

Tecnologias Utilizadas:

-Java 21

-Spring Boot 4.0.2

-Spring Data JPA: Para persistência de dados.

-MySQL: Banco de dados relacional.

-SpringDoc OpenAPI (Swagger): Para documentação e testes da API.

para acessar a API:
    
    http://localhost:8080/swagger-ui.html


----------------------------------------------------------------------------------------

Funcionalidades:

-Listar itens: Retorna todos os produtos cadastrados no mercado.

-Buscar por ID: Localiza um item específico.

-Adicionar item: Cadastra novos produtos com nome, estoque e preço.

-Atualização parcial (Patch): Permite atualizar apenas campos específicos de um item.

-Remover item: Exclui um produto do sistema.
